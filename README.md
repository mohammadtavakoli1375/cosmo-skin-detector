# Cosmo Skin Detector

A Progressive Web App for skin type detection and cosmetic product recommendations.

## Features

- 💄 Skin type detection
- 📱 PWA support
- 🎨 Persian font support (Kalameh)
- 📲 Mobile optimized
- 🚀 Fast and responsive

## Deployment

This project is deployed on Vercel.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
